# Stephen Mar — Personal Brand Website

A premium, minimalist personal brand site for Stephen Mar — entrepreneur and investor in crypto, DeFi, and Web3.

## Features
- Soft off-white + warm tan palette
- External `styles.css` for cleaner, more maintainable premium styling
- Perfectly fitted circular headshot (full visibility, no cropping)
- Elegant cards with hover lifts, refined spacing, and smooth transitions
- Fully responsive with Tailwind + custom CSS

## Deployment (GitHub Pages)
1. Upload `index.html`, `styles.css`, the `assets/images/headshot.jpg` folder, and `README.md`.
2. Go to **Settings → Pages** → Deploy from main branch → / (root).
3. Clear browser cache after deploy for best results.
