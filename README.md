# Stephen Mar — Personal Website

## Deployment (GitHub Pages)

1. Create a new repository
2. Upload:
   - index.html
   - assets/images/headshot.jpg
3. Go to Settings → Pages
4. Select:
   - Branch: main
   - Folder: /root
5. Save

Your site will be live at:
https://yourusername.github.io/repository-name/
